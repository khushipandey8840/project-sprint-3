# Weather App Prompts

Yahan wo commands ya prompts hain jo main is project ko aur behtar banane ke liye use kar sakti hoon:

## 1. Feature: Humidity aur Wind Speed
"Mere weather app mein temperature ke saath-saath 'Humidity' aur 'Wind Speed' bhi add kardo."

## 2. Feature: Background Dynamic Change
"Weather ke condition (jaise Rain, Clear, Clouds) ke hisaab se mera background color apne aap change hona chahiye. Iske liye JavaScript mein logic add karo."

## 3. Feature: Location Button
"Ek 'Current Location' ka button banao, jisse click karte hi user ki location ka weather auto-detect ho jaye."

## 4. Feature: Error Handling
"Agar user galat city name dale, toh alert ki jagah screen par ek sundar error message dikhao."

## 5. Design: Dark Mode
"Is app mein ek 'Dark Mode' toggle switch add karo taaki user light aur dark theme mein switch kar sake."
