# Weather App 🌦️

Ek simple aur interactive Weather App jo OpenWeatherMap API ka use karke real-time weather data dikhata hai.

## Kaise Run Karein
1. Apne folder mein `index.html` file ko open karein.
2. Search bar mein kisi bhi city ka naam likhein (e.g., Delhi, Mumbai).
3. 'Search' button par click karein.

## Features
- Real-time weather update.
- Temperature in Celsius.
- Weather conditions (Sunny, Cloudy, etc.).
- Weather icons.

## Tech Stack
- HTML5
- CSS3
- JavaScript (Fetch API)
